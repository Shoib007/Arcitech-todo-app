# How to Install
### Clone Repo
```
git clone https://github.com/Shoib007/Arcitech-todo-app.git
```
### Backend Installetion
```
cd TaskBackend
pip install -r requirements.txt
python3 manage.py makemigration
python3 manage.py migrate
python3 runserver
```
### Frontend Installetion
```
cd TaskFrontend
npm i
npm run dev
```
